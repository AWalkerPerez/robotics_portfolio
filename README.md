## üöÄ How to View the Portfolio

This portfolio is built as a single-page application using HTML, modern CSS (Tailwind CSS), and JavaScript. It does not require any backend server or special dependencies.

If you wish to examine the source code, download the project assets, and run the portfolio locally on your machine, follow these steps:

1.  **Clone or Download:** Click the **"Code"** button above and select **"Download ZIP"** to get all project files.
2.  **Unzip the Folder:** Extract the contents of the downloaded ZIP file to a location on your computer.
3.  **Open the Portfolio:** Locate the main file, **`index.html`**, in the root of the unzipped folder.
4.  **Run:** Double-click **`index.html`**.

This action will open the portfolio website instantly in your default web browser, displaying the full content and navigation bar exactly as it appears online.
